# Türkçe CloudStream Eklentisi

Bu eklenti FilmEkseni ve DiziDay sitelerini CloudStream uygulamasına entegre eder.

## Özellikler

- FilmEkseni.net film desteği
- DiziDay.de dizi desteği
- Türkçe altyazı desteği
- Chromecast desteği
- İndirme desteği
